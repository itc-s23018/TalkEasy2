# TalkEasy

音声認識技術や音声合成技術を活用し、聴覚に障がいを持つ方々の円滑なコミュニケーションを支援するためのアプリケーションです。

## 主な機能

- **音声の文字化**: マイクからの音声をリアルタイムでテキストに変換して表示します。
- **テキストの音声化**: キーボードで入力した文字を音声で読み上げます。
- **用語データベース**: 会話でよく使われる専門用語や固有名詞を登録・管理できます。

### AIアシスト機能

- **音声入力の補正**: AIが音声認識結果の誤りを自動で補正し、より正確なテキストを生成します。
- **返答の提案**: 会話の流れをAIが解析し、適切な返答の候補を提案します。
- **会話中の用語抽出**: AIが会話の中から重要なキーワードを自動で抽出し、ハイライトします。

## 開発環境

- **IDE**: Android Studio
- **言語**: Kotlin
- **UI**: Jetpack Compose
- **Min SDK**: 26
- **Target SDK**: 36
